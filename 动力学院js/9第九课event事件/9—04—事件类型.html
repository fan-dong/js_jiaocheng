一、鼠标类型事件 
	1.	click		单击鼠标时触发
	2.	dblclik 	双击鼠标时触发
    3.	mousedown 	单击任意一个鼠标按键时触发
    4.	mouseup		鼠标按键被松开
    4.	mouseout 	鼠标从某元素
    5.	mousemove	鼠标在元素之上移动时
    6.	mouseover	鼠标移到某元素之上
说明：鼠标事件对几乎所有元素均可用。


二、键盘事件
	1.	keydown		某个键盘按键被按下
    2.	keypress	某个键盘按键被按下并松开
    3.	keyup		某个键盘按键被松开


三、表单事件
	1.	change
    2.	focus
    3.	blur
    4.	select
    5.	submit
   

四、其他事件
	1.	load
    2.	DOMContentLoaded
    3.	readystatechange
    4.	contextmenu
   	5.	error
   
说明各个事件举例见文件夹：事件类型案例
